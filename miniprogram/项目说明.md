# 智能博客系统 - 微信小程序

## 📱 项目简介

这是智能博客系统的微信小程序版本，与 Web 端功能保持一致，提供原生小程序体验。

## ✨ 主要特性

- 🎨 **紫色渐变主题** - 与 Web 端样式统一
- 📝 **文章系统** - 浏览、搜索、详情、评论
- 👤 **用户系统** - 登录、个人中心、关注
- 💬 **社交功能** - 评论、私信、通知
- 👑 **VIP 会员** - 会员特权、文章下载
- 🤖 **AI 聊天** - 智能对话功能
- 📱 **自定义 TabBar** - 使用 Emoji 图标

## 🚀 快速开始

### 1️⃣ 关闭域名校验（重要！）

**必须先完成这一步，否则无法访问 localhost**

```
微信开发者工具 → 详情 → 本地设置 
→ ☑️ 不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书
```

详细说明：[如何关闭域名校验.txt](./如何关闭域名校验.txt)

### 2️⃣ 配置 AppID

修改 `project.config.json`：
```json
{
  "appid": "your-appid-here"
}
```

### 3️⃣ 启动后端服务

确保后端服务运行在 `http://localhost:8080`

### 4️⃣ 导入项目

```
微信开发者工具 → 导入项目 → 选择 miniprogram 目录
```

### 5️⃣ 编译运行

点击"编译"按钮，即可在模拟器中预览

## 🔧 配置说明

### 开发环境

默认使用 `http://localhost:8080`，无需修改。

### 生产环境

修改 `utils/env.js` 中的 `prodConfig`：

```javascript
const prodConfig = {
  BASE_URL: 'https://your-domain.com/api',
  WS_URL: 'wss://your-domain.com/ws'
}
```

### 环境自动切换

- 开发者工具：自动使用开发环境配置
- 真机预览/正式版：自动使用生产环境配置

## ⚠️ 注意事项

1. **域名校验**
   - 开发阶段必须关闭域名校验
   - 真机调试需要 HTTPS
   - 上线前必须配置合法域名

2. **后端接口**
   - 需要支持微信登录接口
   - 所有接口需要支持 CORS
   - 生产环境必须使用 HTTPS

3. **小程序限制**
   - 主包大小不超过 2MB
   - 总包大小不超过 20MB
   - 使用分包加载优化

## 🐛 常见问题

### Q: 域名校验错误？
A: 查看 [如何关闭域名校验.txt](./如何关闭域名校验.txt)

### Q: 登录失败？
A: 确保后端支持 `/api/auth/wx-login` 接口

### Q: 真机无法访问？
A: 真机必须使用 HTTPS，可以使用内网穿透工具
