## ⚠️ 你需要做的（重要！）

### 必须执行：构建 npm

在微信开发者工具中：

```
1. 点击顶部菜单：工具 → 构建 npm
2. 等待构建完成（几秒钟）
3. 看到 miniprogram_npm 目录生成即成功
```

**为什么必须构建？**
- npm 包需要转换为小程序可识别的格式
- 构建后会生成 `miniprogram_npm` 目录
- 不构建的话图标无法显示

### 可选：启用 npm 模块

在微信开发者工具中：

```
1. 点击右上角"详情"
2. 选择"本地设置"
3. 勾选"使用 npm 模块"
```

## ❌ 如果图标不显示

### 方法 1：重新构建
1. 工具 → 构建 npm
2. 清缓存 → 全部清除
3. 重新编译

### 方法 2：检查配置
1. 确认 `miniprogram_npm` 目录存在
2. 确认"使用 npm 模块"已勾选
3. 查看控制台错误信息

### 方法 3：重启工具
1. 关闭微信开发者工具
2. 重新打开项目
3. 重新编译

## 🔄 如何更换其他图标

1. 访问 [Vant Weapp 图标列表](https://vant-contrib.gitee.io/vant-weapp/#/icon)
2. 选择你喜欢的图标
3. 修改 `custom-tab-bar/index.js` 中的图标名称
4. 重新编译查看效果

示例：
```javascript
{
  pagePath: "/pages/index/index",
  text: "首页",
  icon: "home-o",      // 改为你想要的图标
  selectedIcon: "home" // 改为对应的实心图标
}
```
