# 快速启动指南

## 🚀 立即开始

### ⚠️ 重要：首次运行必读

**如果看到域名校验错误，请按以下步骤操作：**

1. 点击微信开发者工具右上角的 **"详情"** 按钮
2. 切换到 **"本地设置"** 标签页
3. 勾选 **"不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"**
4. 重新编译项目

详细说明请查看：[开发环境配置指南.md](./开发环境配置指南.md)

---

### 1. 配置后端地址

打开 `utils/config.js`，修改 API 地址：

```javascript
// 项目已配置自动环境切换
// 开发环境自动使用 localhost
// 生产环境需要修改 utils/env.js 中的 prodConfig
```

如需修改开发环境地址，编辑 `utils/env.js`：

```javascript
const devConfig = {
  BASE_URL: 'http://localhost:8080/api',  // 修改为你的后端地址
  WS_URL: 'ws://localhost:8080/ws'
}
```

### 2. 配置小程序 AppID

打开 `project.config.json`，修改 appid：

```json
{
  "appid": "your-appid-here"  // 改为你的小程序 AppID
}
```

### 3. 导入项目

1. 打开微信开发者工具
2. 点击"导入项目"
3. 选择 `miniprogram` 目录
4. 填入 AppID
5. 点击"导入"

### 4. 编译运行

导入成功后，点击"编译"按钮即可在模拟器中预览。

## 🔧 常见问题

### Q1: 编译报错 "不在合法域名列表中"
**A:** 这是正常的开发阶段提示。解决方法：
1. 点击右上角"详情"
2. 切换到"本地设置"
3. 勾选"不校验合法域名..."
4. 重新编译

详细说明：[开发环境配置指南.md](./开发环境配置指南.md)

### Q2: 如何测试登录功能？
**A:** 需要后端支持微信登录接口 `/api/auth/wx-login`，接收 `code` 参数并返回 `token`。

### Q3: 如何配置服务器域名？
**A:** 在小程序后台"开发 > 开发管理 > 开发设置"中配置：
- request 合法域名：`https://your-backend-url.com`
- socket 合法域名：`wss://your-backend-url.com`

### Q4: 真机调试时无法请求接口？
**A:** 检查以下几点：
1. 后端地址必须是 HTTPS
2. 域名已在小程序后台配置
3. 关闭"不校验合法域名"选项（开发阶段可开启）
