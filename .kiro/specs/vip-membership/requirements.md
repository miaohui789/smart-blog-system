# VIP 会员功能需求规格

## 概述
为博客系统添加 VIP 会员功能，用户可通过激活密钥晋升为 VIP 身份，享受专属特权。

## 用户故事

### US-001: VIP 密钥激活
**作为** 普通用户  
**我想要** 通过输入有效的激活密钥来激活 VIP 身份  
**以便于** 享受 VIP 专属特权

**验收标准:**
- [ ] 用户可在 VIP 激活页面输入密钥
- [ ] 密钥格式为 `VIP-XXXX-XXXX-XXXX-XXXX`
- [ ] 有效密钥激活后，用户立即获得 VIP 身份
- [ ] 已使用或无效密钥显示相应错误提示
- [ ] 已是 VIP 用户再次激活时延长到期时间

### US-002: VIP 信息查看
**作为** VIP 用户  
**我想要** 查看我的 VIP 会员信息  
**以便于** 了解我的会员等级和到期时间

**验收标准:**
- [ ] 个人中心显示 VIP 名片（等级、到期时间）
- [ ] 显示今日剩余权益（下载次数、加热次数）
- [ ] 显示 VIP 特权说明

### US-003: 文章 Markdown 下载
**作为** VIP 用户  
**我想要** 下载文章的 Markdown 源文件  
**以便于** 离线阅读或学习

**验收标准:**
- [ ] VIP 用户在文章详情页可见下载按钮
- [ ] 非 VIP 用户不显示下载按钮或显示 VIP 专属提示
- [ ] 下载次数受 VIP 等级限制（普通5次/日，高级20次/日，超级无限）
- [ ] 下载次数用尽时显示提示

### US-004: 文章加热
**作为** VIP 用户  
**我想要** 为喜欢的文章增加热度  
**以便于** 帮助优质文章获得更多曝光

**验收标准:**
- [ ] VIP 用户在文章详情页可见加热按钮
- [ ] 加热次数受 VIP 等级限制（普通3次/日，高级10次/日，超级30次/日）
- [ ] 不同等级加热值不同（普通+10，高级+30，超级+100）
- [ ] 加热后文章热度值增加
- [ ] 加热次数用尽时显示提示

### US-005: VIP 标识显示
**作为** 网站访客  
**我想要** 在评论区看到 VIP 用户的专属标识  
**以便于** 识别 VIP 用户

**验收标准:**
- [ ] VIP 用户评论时头像旁显示 VIP 图标
- [ ] 不同等级显示不同颜色（普通铜色，高级银色，超级金色）

### US-006: 管理员密钥管理
**作为** 管理员  
**我想要** 管理 VIP 激活密钥  
**以便于** 控制 VIP 会员的发放

**验收标准:**
- [ ] 可批量生成指定等级和有效期的密钥
- [ ] 可查看所有密钥列表（支持搜索、筛选、分页）
- [ ] 可禁用/启用密钥
- [ ] 可删除密钥
- [ ] 可查看密钥使用情况

### US-007: 管理员会员管理
**作为** 管理员  
**我想要** 管理 VIP 会员  
**以便于** 维护会员信息

**验收标准:**
- [ ] 可查看所有 VIP 会员列表
- [ ] 可编辑会员等级和到期时间
- [ ] 可删除会员 VIP 身份
- [ ] 可查看 VIP 统计数据（总数、各等级分布、激活趋势）

## 技术规格

### 数据库变更
参考: #[[file:docs/VIP会员功能开发文档.md]]

需要执行的 SQL:
1. `sys_user` 表添加 VIP 相关字段
2. 创建 `vip_member` 表
3. 创建 `vip_activation_key` 表
4. 创建 `vip_article_heat` 表
5. `blog_article` 表添加 `heat_count` 字段
6. 插入 VIP 管理菜单

### API 接口

#### 用户端
- `POST /api/vip/activate` - 激活 VIP
- `GET /api/vip/info` - 获取 VIP 信息
- `GET /api/vip/privileges` - 获取 VIP 权益说明
- `POST /api/vip/heat/{articleId}` - 文章加热
- `GET /api/articles/{id}/download` - 下载文章 MD

#### 管理端
- `GET /api/admin/vip/members` - 会员列表
- `PUT /api/admin/vip/members/{id}` - 编辑会员
- `DELETE /api/admin/vip/members/{id}` - 删除会员
- `GET /api/admin/vip/keys` - 密钥列表
- `POST /api/admin/vip/keys/generate` - 批量生成密钥
- `PUT /api/admin/vip/keys/{id}/status` - 禁用/启用密钥
- `DELETE /api/admin/vip/keys/{id}` - 删除密钥
- `GET /api/admin/vip/statistics` - VIP 统计数据

### VIP 等级权益

| 等级 | 名称 | 下载次数/日 | 加热次数/日 | 加热值 | 专属标识 |
|------|------|------------|------------|--------|---------|
| 1 | 普通 VIP | 5 | 3 | +10 | 铜色 |
| 2 | 高级 VIP | 20 | 10 | +30 | 银色 |
| 3 | 超级 VIP | 无限 | 30 | +100 | 金色 |
