# 新功能开发说明

## 功能概述

### 功能1：用户详情页
- 点击评论区头像/用户名 → 跳转用户详情页
- 用户详情页展示：用户资料 + 发布的文章列表
- 支持文章排序（最新/阅读/点赞/评论）、关键词搜索
- 返回上一页按钮
- 主页右上角下拉菜单可查看"我的主页"

### 功能2：关注 + 私信系统
- 关注/取消关注用户
- 我的关注页面
- 实时私信功能（WebSocket推送）

## API 接口

### 用户详情
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/user/{userId}/profile | 获取用户详情 |
| GET | /api/user/{userId}/articles | 获取用户文章列表 |
| POST | /api/user/{userId}/follow | 关注用户 |
| DELETE | /api/user/{userId}/follow | 取消关注 |
| GET | /api/user/following | 获取我的关注列表 |
| GET | /api/user/followers | 获取我的粉丝列表 |
| GET | /api/user/{userId}/following | 获取指定用户的关注列表 |
| GET | /api/user/{userId}/followers | 获取指定用户的粉丝列表 |

### 私信
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/message/send | 发送私信 |
| GET | /api/message/conversations | 获取会话列表 |
| GET | /api/message/conversations/{id} | 获取会话消息 |
| POST | /api/message/conversations/{id}/read | 标记已读 |
| GET | /api/message/unread-count | 获取未读数 |
| GET | /api/message/conversation-with/{userId} | 获取与指定用户的会话 |

### 数据库


### 前端新增文件
```
api/notification.js           -- 通知API
stores/notification.js        -- 通知状态管理
utils/websocket.js            -- WebSocket工具（通知+评论实时刷新）
views/Notification/index.vue  -- 通知页面
```

### 通知类型
| 类型 | 说明 |
|------|------|
| LIKE_ARTICLE | 文章被点赞 |
| FAVORITE_ARTICLE | 文章被收藏 |
| COMMENT | 文章被评论 |
| REPLY | 评论被回复 |
| FOLLOW | 被关注 |
| SYSTEM | 系统通知 |

### 通知API
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/notifications | 获取通知列表 |
| GET | /api/notifications/unread-count | 获取未读数量 |
| POST | /api/notifications/{id}/read | 标记单条已读 |
| POST | /api/notifications/read-all | 标记全部已读 |
| DELETE | /api/notifications/{id} | 删除通知 |
| DELETE | /api/notifications/clear | 清空通知 |

### 实时功能
- 评论区实时刷新：用户发表评论后，其他正在浏览该文章的用户会实时看到新评论
- 通知实时推送：点赞、收藏、评论、回复、关注等操作会实时推送通知给相关用户
