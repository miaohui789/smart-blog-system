# 邮箱登录功能说明

## 功能概述

实现了三种登录方式，用户可以灵活选择：

1. **用户名密码登录** - 传统的用户名+密码登录方式
2. **邮箱密码登录** - 使用邮箱地址+密码登录
3. **邮箱验证码登录** - 使用邮箱地址+验证码登录（无需记住密码）

## 前端实现

### 文件修改

#### 1. `frontend/user/src/api/user.js`
新增API接口：
- `loginByEmailCode(email, code)` - 邮箱验证码登录
- `sendLoginCode(email)` - 发送登录验证码

#### 2. `frontend/user/src/stores/user.js`
更新 `login` 方法，支持多种登录类型：
- `login(loginForm, loginType)` - loginType可选值：'username'（默认）、'emailCode'

#### 3. `frontend/user/src/views/Login/index.vue`
完全重构登录页面：
- 添加三个登录方式切换Tab
- 每种登录方式独立的表单和验证规则
- 验证码输入框带倒计时功能（60秒）
- 保持原有的高端优雅样式，兼容亮暗色主题

### UI特性

- **Tab切换动画**：平滑的渐变背景和阴影效果
- **验证码倒计时**：发送后60秒内不可重复发送
- **表单验证**：
  - 用户名密码：必填验证
  - 邮箱密码：邮箱格式验证 + 必填验证
  - 邮箱验证码：邮箱格式验证 + 6位数字验证码验证
- **响应式设计**：移动端自适应
- **图标使用**：Element Plus组件图标（User、Message、ChatLineRound、Key等）

#### 4. `backend/src/main/java/com/blog/controller/web/UserController.java`
已有接口（无需修改）：
- `POST /api/auth/login/email` - 邮箱验证码登录
- `POST /api/auth/login/send-code` - 发送登录验证码

更新接口：
- `POST /api/auth/login` - 支持用户名或邮箱+密码登录

### API接口

#### 1. 用户名/邮箱密码登录
```
POST /api/auth/login
Content-Type: application/json

{
  "username": "用户名或邮箱",
  "password": "密码"
}
```

#### 2. 邮箱验证码登录
```
POST /api/auth/login/email?email=xxx@example.com&code=123456
```

#### 3. 发送登录验证码
```
POST /api/auth/login/send-code?email=xxx@example.com
```

## 安全特性

1. **验证码有效期**：验证码5分钟内有效
2. **发送频率限制**：同一邮箱60秒内只能发送一次验证码
3. **账号状态检查**：
   - 已注销账号无法登录
   - 已冻结账号无法登录
   - 未注册邮箱无法发送验证码
4. **角色权限验证**：只有拥有有效角色的用户才能登录

## 使用说明

### 用户端使用

1. 访问登录页面
2. 选择登录方式：
   - **账号登录**：输入用户名和密码
   - **邮箱登录**：输入邮箱地址和密码
   - **验证码登录**：输入邮箱地址，点击"获取验证码"，输入收到的验证码
3. 点击"登录"按钮

### 开发者注意事项

1. 邮箱验证码登录需要配置邮件服务
2. 验证码存储在Redis中，需要确保Redis服务正常运行
3. 前端倒计时仅为用户体验优化，后端仍有频率限制保护

## 兼容性
